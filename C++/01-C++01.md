### 指针

指针就是一个地址。



int * p;//定义指针

p=&a;//将变量 a 的地址给 p

*p=1000;// *p 代表 p 指针指向地址的存储数据



32 位操作系统，指针 4 位

64 位操作系统，指针 8 位



空指针：指向内存中编号为 0 的空间，即 int *p=NULL; 

空指针是不可以访问的：

```c++
int *p=NULL;

*p=100;// 报错
```



野指针

int * p=(int *)0x1100; //瞎找的地址，报错：读取访问权限冲突

常量指针：指针指向可以改，指针的值不可以改

const int *p=&a;

*p=20;//错误

p=&b;//正确



指定常量：指向不可以改，值可以改

int * const p=&a;



const 修辞指定和常量：都不可以改

const int * const p=&a



### 指针与数组

```C++
int* p = arr;//arr 就是数组首地址
cout << *p << endl;
p++;//让指针向后偏移 4 个字节
cout << *p << endl;
//遍历
for (int i = 0;i < 5;i++) {
		cout << *p << endl;
		p++;
	}
```
